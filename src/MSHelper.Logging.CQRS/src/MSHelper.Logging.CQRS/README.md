# MSHelper.Logging.CQRS : Centralized logging for CQRS with Serilog and Seq.
:star: Star us on GitHub — it motivates us a lot!


## Installation

This document is for the latest MSHelper.Logging.CQRS **1.0.0 release and later**.

`dotnet add package MSHelper.Logging.CQRS`

## Dependencies

-- [MSHelper](https://www.nuget.org/packages/MSHelper)

-- [MSHelper.Logging](https://www.nuget.org/packages/MSHelper.Logging)

-- [MSHelper.CQRS.Queries](https://www.nuget.org/packages/MSHelper.CQRS.Queries)

-- [MSHelper.CQRS.Commands](https://www.nuget.org/packages/MSHelper.CQRS.Commands)

-- [MSHelper.CQRS.Events](https://www.nuget.org/packages/MSHelper.CQRS.Events)
